# Installation
<b>Manual: </b>
```bash
apt update && apt install git python3 -y && git clone https://github.com/sqlmerr/hurricane && cd hurricane && pip3 install -r req* && python -m hurricane
```

<b>Using docker compose: </b>
```base
docker compose up --build -d
```
